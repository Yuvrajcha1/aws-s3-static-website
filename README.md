# 🚀 AWS S3 Static Website Hosting Project

## 📌 Overview
This project demonstrates how to host a static website using Amazon S3 and distribute it globally using CloudFront.

---

## 🛠️ Services Used
- Amazon S3
- AWS CloudFront
- IAM (Bucket Policy)
